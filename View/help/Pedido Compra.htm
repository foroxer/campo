Pedidos Compras

Dentro de esta pantalla se podrá ir cargando los faltantes de mercadería para que después se tome este listado y se compre lo que hay en el mismo, para una mejor organización. Funciona de la siguiente forma:

  1. Se elige que artículo es el faltante y se desea agregar al pedido, ya sea mediante el primer o segundo desplegable, el primero tiene los códigos y el segundo tiene el nombre de los artículos.

  2. Luego se agrega la cantidad que se desea comprar, este es un valor referencial, ya que luego se cargara en el sistema la cantidad exacta que llego con el pedido

  3. Se puede ver que debajo de esto hay 3 botones los mismos ayudan a cargar, modificar y borrar un artículo del pedido

    1. Cargar: es para ingresar un nuevo pedido de ese articulo

    2. Modificar: es para modificar algo de los pedidos de los pedidos ya cargados

    3. Borrar: es para eliminar algunos de los artículos ya cargados, por si se cargó por error

  4. Luego a la izquierda se puede observar que hay otros 3 botones, cerrar pedido, mostrar pedidos cerrados e imprimir en PDF

    1. Cerrar pedido: cuando se cliquea se abrirá una ventana pequeña la cual servirá para elegir el lugar donde queremos guardar el pedido. El mismo se guardara en XML, que es tipo de archivo que muestra de manera fácil como es el pedido

    2. Mostrar pedidos cerrados:cuando se cliquea también aparecerá una ventana pero esta vez es para elegir el archivo que se querrá abrir y querer que se liste. Cuando esto se haga se listara ese pedido, por lo cual lo que teníamos cargado no se verá pero no es porque se borró, simplemente se muestra el archivo que abrimos

    3. Imprimir en PDF: cuando se cliquee se abrirá una ventana en la cual se seleccionara a donde se quiere guardar el pdf. El mismo tendrá el pedido para que se pueda imprimir y utilizar para hacer el encargue delos productos.
