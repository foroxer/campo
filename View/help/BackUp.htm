BackUp

Dentro de esta pantalla se podrá hacer el BackUp del sistema, cada vez que se crea pertinente

  1. Para comenzar primero se debe cliquear el botón “Examinar” el cual abrirá una ventana en la cual hay que elegir donde queremos guardar el BackUp, en el campo de Ruta aparecerá la dirección que elegimos, la misma se puede modificar si se cliquea el botón “Modificar”

  2. Luego se debe poner un nombre en el campo Nombre el cual hará que podamos identificar más fácilmente el BackUp en un futuro.

  3. Por ultimo hay que completar el campo de Cantidad de copias, lo que hará es que divide nuestro BackUp en la cantidad que pongamos, lo que hace es que sea más fácil de guardar y que sea más transportable

  4. Para finalizar apretamos el botón “Realizar BackUp”, lo que hará es guardar el BackUp en la dirección que elegimos con el nombre que elegimos.

De esta forma el BackUp quedara guardado en el sistema.
