Generar Contraseña

Esta pantalla se usa para realizar el cambio de contraseña, ya sea que se ha olvidado la contraseña o por si quiere cambiar la clave.

Primero cuando se abre la pantalla se verá dos campos uno de Nick y otro de Mail, los mismos deben ser llenados haciendo referencia al usuario al que se le quiere cambiar la contraseña. Luego se deberá cliquear el botón “Verificar” para ver si los datos ingresados son correctos. Si así es, los controles cambiaran y aparecerán otros. Si los datos ingresados no son correctos, tiene 3 intentos, después se bloqueara el usuario, y deberá pedir ayuda al administrador para que se desbloquee ese usuario.

Ahora aparece un campo de contraseña, la cual se puede generar de dos formas, o se cliquea el botón “Generar Contraseña” y genera una contraseña aleatoria o si quiere agregar una contraseña personalizada puede ingresar en el campo la contraseña que desee, si posiciona el mouse sobre el campo, le dará las indicaciones que debe cumplir la contraseña para poder ser válida.

Por último se cliquea el botón “Confirmar Cambio” y la contraseña será guardada
