LogIn

En esta pantalla podrá ver el LogIn del programa, acá va una mini explicación de los pasos que deberá seguir para poder loguearse con éxito

    1. Cliquear el botón Generar Conexión, el mismo mostrara nuevos botones, dentro de ellos deberá hacer lo siguiente:

      a. Primero ver el desplegable y seleccionar la base de datos que corresponda, la que deba seleccionar la sabrá porque se lo dirá el administrador o la persona que explique el funcionamiento de la aplicación

      b. Cliquear el botón “Generar String” para que se plasme lo que se seleccionó recién, apretar Aceptar

      c. Luego apretar el botón “Probar conexión” el mismo funciona para comprobar si está bien la conexión con la base de datos , apretar Aceptar

      d. Por ultimo cliquear Aceptar

      e. Lo llevara nuevamente a los botones que tenía en un principio

    2. Deberá ingresar su Nick y contraseña, los mismos los podrá ingresar mal hasta 3 veces, después quedara bloqueado y deberá pedir ayuda al administrador

    3. Apretar Ingresar para poder ingresar al sistema, si está todo bien le deberán aparecer dos mensajes que los mismos dicen que el sistema está bien, si sale algún mensaje de error debe contactar al administrador para pedir ayuda

Botones extras:

  • “He olvidado la contraseña” el mismo sirve para no sabe la contraseña, si lo cliquea le aparecerá una ventana que lo ayudara a cambiar la clave, la misma esta explicada dentro de los documentos en Seguridad - Generar contraseña

  • Botón “Idioma” el mismo sirve para cambiar desde el inicio el idioma al sistema, se abrirá una ventana, la misma esta explicada dentro de los documentos en Idioma - Cambiar Idioma
