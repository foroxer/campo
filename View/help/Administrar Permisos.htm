Administrar Permisos

Dentro de esta pantalla se pueden realizar varias opciones. Se puede observar en la parte superior 3 pestañas:

  • Familia

  • Patente

  • Administrar Permisos

    • A continuación se explicara una por una

    Familia

    En esta pestañase podrá modificar a las familias

  • Se podrá dar de altas a nuevas familias, esto sucede si se completa solamente el campo nombre, y cliqueando el botón “Alta”, de esta forma se habrá dado de alta a una familia nueva

  • Se podrá modificar familias ya existentes, de dos formas. La primera es cliquear desde el listado la familia que queremos modificar. El campo nombre se llenara con el nombre de la familia elegido y el otro campo el Nombre Nuevo es el que debe llenar uno, el nombre puesto en ese campo es el que quedara guardado en el sistema. La segunda opción es completar a mano el campo nombre con el nombre correspondiente a la familia que queremos modificar y después proseguir de la misma manera, llenar el campo nombre nuevo. Por ultimo darle cliquear el botón “Modificar” para que quede guardado el cambio

  • Se podrá dar de baja a una familia, solamente cuando se cliquea desde el listado la familia a la que le queremos dar de baja y luego presionamos el botón “Baja” de esta forma la familia será eliminada

    • Patente

    En esta pestaña se podrá agregar o quitar patentes del sistema.

  • Para poder cargarlas primero se debe elegir una opción del desplegable de permisos, ahí se mostraran los permisos que hay en el sistema, luego se debe completar el campo de nombre patente, el mismo debe ser un nombre por el cual se pueda identificar a esa patente, puede ser el mismo o se puede cambiar. Una vez hecho todo eso, se cliquea el botón “Cargar” y se guardaran los cambios

  • Para poder quitar una patente del sistema lo único que hay que hacer es seleccionar una opción del despegable de permisos y apretar el botón “Quitar” de esta forma la patente será borrada del sistema

    • Administrar Permisos

    En esta pestaña se administrara la relación entre las familias con las patentes o familias con familias. El listado que está a la derecha mostrara las relaciones entre familia y patentes, se mostrara en forma descendente mostrando que contiene cada uno, esto se actualizara a medida que se vayan cargando relaciones nuevas y mostrara diferentes relaciones dependiendo de qué familia se seleccione. En la parte inferior se ve otro listado, que muestra lo mismo que a la derecha pero en forma de tabla, en este caso se mostrara todo el tiempo, y se actualizara cuando se agreguen o borren relaciones.

  • Para agregar familias a familias:

    1. Primero se selecciona una familia del despegable familia y se debe apretar el botón “Seleccionar”

    2. Luego del mismo despegable se selecciona otra familia y en vez de apretar el botón “Seleccionar” se cliquea el botón “Agregar” que está a la altura de familia, esto lo que hará es que una familia esté relacionada con una familia

    3. Para guardar esto se debe cliquear el botón “Cargar”

    4. Para borrar relaciones familia - familia, se debe hacer un proceso muy parecido, se hace todo tal cual, la única diferencia que al final se aprieta el botón “Borrar” en vez del que “Agregar” de esta forma la relación familia – familia será borrada

  • Para agregar patentes a familias:

    1. Primero se selecciona a la familia de la misma forma que arriba, primero se elige del despegable a la familia que queremos y ponemos “Seleccionar”

    2. Del despegable de debajo de patentes se selecciona a la patente a la que queremos relacionar con esa familia

    3. Se aprieta el botón “Agregar” pero el que está a la altura de la patente

    4. Y por último se cliquea el botón “Cargar”. De esta forma quedara relacionada la familia con la patente

    5. Para darle de baja a una relación es lo mismo que los pasos anteriores pero en vez de apretar el botón “Agregar” se cliquea el botón “Borrar”, y luego el botón “Cargar” de esta forma quedara eliminada esa relación.

    Sea cual sea la relación que se desea crear o eliminar todos los movimientos se verán reflejados en las dos tablas, en la de debajo de forma permanente, todos los movimientos se verán y en la de la derecha solo se verán las relaciones de la familia seleccionada
