Comprar

En esta pantalla se mostrara todo lo necesario para realizar una compra. Para este paso se deberá estar con el pedido ya comprado y solo cargarlo en el sistema para poder aumentar los stocks de los productos y poder dejar asentada la compra, junto con el proveedor al que se le compro. Si no sabe cómo comenzar a continuación se explicara el paso a paso de cómo generar una compra.

  1. Primero se deberá seleccionar el proveedor, ya sea mediante el primer o segundo desplegable que hace referencia a proveedor, el primero mostrara los CUIT de todos los proveedores, el segundo mostrara el nombre de todos los proveedores.

  2. Una vez que se seleccione el proveedor se deberá cliquear el botón “Generar Compra” el mismo generara un numero de compra para poder comenzar la compra, además, el mismo desaparecerá una vez que sea cliqueado para que no se pueda volver a cliquear

  3. Se deberá cliquear el botón “Cargar Detalle”, el mismo habilitara los botones necesarios para que se pueda ir cargando los detalles de la compra

  4. Para poder cargar los detalles se deberá cliquear el botón “Traer Pedido” el mismo hará que aparezca una ventana pequeña en la cual se deberá elegir en qué lugar de la computadora se aloja el documento de pedido que se desea comprar, una vez seleccionado el archivo se le dará Aceptar

  5. Se cargara en el listado de la derecha todo lo que está en el pedido que se debe comprar / cargar en el sistema para que se pueda aumentar el stock de los productos.

  6. Para poder darle de alta a los detalles se deberá ir cliqueando las filas de la lista y se cargaran los datos en el detalle para que se le pueda dar de alta, el valor que se debe agregar es la cantidad, ya que lo que aparece es la cantidad que se quería, pero puede llegar menos, más o igual cantidad de la mercadería pedida.

  7. Y así se debe hacer con todas las filas que hay en la lista. Mientras que se hace eso se puede modificar o dar de baja a un detalle que hayamos cargado mal.

  8. Punto a parte: Si se debe dar de alta a un nuevo artículo, se podrá cliquear el botón “Cargar nuevo artículo” que el mismo abrirá la ventana de articulo para poder darle de alta, ese funcionamiento se explica en el documento llamado “Articulo”

  9. Continuando, cuando se finalice de darle de alta a todos los detalles se debe cliquear “Cerrar detalle” el cual deshabilita los botones para crear detalles

  10. Si se cliquea sin querer se puede cliquear “Editar detalle” que vuelve todo un paso para atrás y habita todos los botones anteriores para poder seguir editando los detalles

  11. Cuando realmente se finalice todo y se haya cliqueado el botón “Cerrar detalle” se podrá observar el total de la compra y se debe cliquear el botón “Comprar” para que la compra finalice. El mismo mostrara un cartel preguntando si está seguro que desea comprar

    1. Si se cliquea que sí, mostrara nuevamente una ventana pequeña en la cual se deberá elegir el lugar donde se quiere guardar el comprobante de compra. y luego cliquear Aceptar y mostrara a los segundos un mensaje diciendo PDF guardado

    2. Si se cliquea que no, volverá todo como estaba

  12. Por ultimo si se desea salir de la pantalla de compra y cancelar la compra se podrá hacer cliqueando la X posicionada en la parte superior del lado derecho
