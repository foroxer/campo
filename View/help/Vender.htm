Vender

Dentro de esta pantalla se podrá cargar la compra que haga un cliente, junto con los productos que se vendieron, y las cantidades dejando plasmado en el sistema esa venta

  1. Se deberá seleccionar al cliente, ya sea por el primer o segundo desplegable que hace referencia al mismo, el primero muestra los DNI y el segundo muestra los nombres

  2. Una vez seleccionado se deberá seleccionar el botón “Generar Venta” para que se cree el número de venta

  3. A continuación, se deberá cliquear el botón “Cargar detalle” el mismo habilitara lo necesario para poder cargar, modificar y borrar los detalles de la venta.

  4. Luego se podrá ir agregando los productos que compro el cliente, junto con su cantidad, primero deberá seleccionar el producto mediante los desplegables, el primero para los códigos y el segundo para los nombres. El precio no se podrá modificar. La cantidad se agrega la que fue vendida. Y el subtotal se va a ir actualizando a medida que se agreguen nuevos detalles

  5. Cuando se termine de cargar todo se cliqueara el botón “Cerrar Detalle” el mismo deshabilitara lo de detalle y habilitara el botón “Vender”. Además, se mostrara el total de la venta, y la totalidad de los descuentos.

  6. Si se quiere modificar el detalle, se deberá cliquear el botón “Editar Detalle”, y cuando se termina cliquear el botón “Cerrar Detalle”

  7. Para finalizar, se deberá cliquear el botón “Vender”, el mismo mostrara un mensaje con el importe total

    1. Si se acepta muestra una ventana en la cual se deberá elegir donde guardar el comprobante para dárselo al cliente

    2. Si se da cancelar, se vuelve al estado anterior y se puede seguir vendiendo

  8. En cambio, si se quiere cancelar la venta en cualquier momento se puede apretar la cruz y sale un mensaje preguntando si está seguro de cancelar la compra
