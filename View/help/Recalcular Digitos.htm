Recalcular Digitos

Esta ventana solo estará disponible de ser necesaria su utilización ya que solo la podrá utilizar el administrador una vez que el programa tenga algún problema

Dentro de esta ventana se mostrara cuáles son los campos que poseen dígitos verificadores, cada una con un botón, si esa entidad tiene un error el campo mostrar error y estará habilitada el botón, si así es, deberá cliquear el botón habilitado. Cuando todos los botones estén deshabilitados nuevamente, deberá cliquear el botón Recalcular DVV para poder terminar con el proceso de recalcular dígitos
